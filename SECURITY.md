# Security Policy

## Supported Versions

We currently provide security updates for the following versions:

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.5.x   | :x:                |

## Reporting a Vulnerability

**Please do not report security vulnerabilities through public GitHub issues.**

If you discover a potential security vulnerability, please send an email to **r230302@famt.ac.in**. 

### Our Process:
1. **Acknowledgement:** You will receive a response within 48 hours.
2. **Investigation:** We will investigate the issue and keep you updated on the progress.
3. **Resolution:** Once fixed, we will credit you in our release notes (unless you prefer to remain anonymous).
